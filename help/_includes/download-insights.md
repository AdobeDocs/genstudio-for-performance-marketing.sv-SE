# Hämta insikter

Klicka på nedladdningsikonen (pilen pekar nedåt) ovanför den högra sidan av tabellen för att ladda ned en CSV-fil baserat på den visningsbara tabellen. Hämtningen startar automatiskt och placerar CSV-filen på din standardplats för hämtning. Vissa tabeller kan ha flera sidor, som du ser nedanför den högra sidan av tabellen. CSV-filen innehåller data från _alla_ sidor i tabellen.
