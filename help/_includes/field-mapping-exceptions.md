# Undantag för fältmappning

Vissa kanaler har bara stöd för att ersätta specifika fält. Exempel:

* Meta- och LinkedIn-annonser ersätter rubriken, brödtexten och introduktionstexten i den omgivande fönsterstandarden, inte i själva annonsen. Överväg att mappa text på bilden som `on_image_text`.
